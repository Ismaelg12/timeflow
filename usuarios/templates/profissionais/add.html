Política de Privacidade e Termos de Uso do Sistema

Ao criar uma conta no TimeFlow, você concorda com os termos estabelecidos abaixo:

1. Proteção de Dados Pessoais (LGPD)

O TimeFlow segue rigorosamente a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018). Seus dados pessoais são coletados e processados exclusivamente para:

  • Identificação e autenticação no sistema
  • Registro e controle de ponto eletrônico
  • Geração de relatórios trabalhistas
  • Comunicação sobre o sistema
2. Coleta e Uso de Dados

Dados coletados: nome completo, CPF, telefone, dados profissionais e registros de ponto.

Finalidade: Os dados são utilizados exclusivamente para gestão de ponto eletrônico e cumprimento de obrigações trabalhistas.

3. Segurança da Informação

Implementamos medidas de segurança para proteger seus dados contra:

  • Acessos não autorizados
  • Alterações indevidas
  • Divulgação não autorizada
4. Responsabilidades do Usuário

Como usuário, você concorda em:

  • Manter suas credenciais de acesso em sigilo
  • Notificar imediatamente em caso de perda ou suspeita de acesso não autorizado
  • Utilizar o sistema apenas para fins legítimos de registro de ponto
  • Fornecer informações verdadeiras e atualizadas
5. Registro de Ponto

O sistema utiliza geolocalização para validar registros de ponto. Ao utilizar o sistema, você autoriza:

  • Coleta de coordenadas GPS para verificação de localização
  • Armazenamento de horários e locais de registro
  • Compartilhamento dessas informações com seu empregador para fins trabalhistas
6. Direitos do Titular

Conforme a LGPD, você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a eliminação de dados tratados com seu consentimento
  • Revogar o consentimento a qualquer momento
Importante: O não cumprimento destes termos pode resultar na suspensão ou cancelamento do acesso ao sistema.
